CLI Reference

Complete command-line interface documentation for Zepplin

Commands

init add update build publish login discover browse trending help
init Project Setup

Initialize a new Zepplin project in the current directory. Creates the configuration files needed to manage dependencies.

zepplin init

Creates zepplin.toml (project configuration) and zepplin.lock (dependency lock file) in the current directory.

add Dependencies

Add a package dependency to your project. The package will be resolved from the configured registry and added to your project configuration.

zepplin add <package> [OPTIONS]

Arguments

<package> Package name, optionally with version (e.g., "zig-clap" or "zap@0.3.0")

Options

--registry <url> Use a custom registry URL instead of the default

Examples

zepplin add zig-clap
zepplin add zap@0.3.0
zepplin add mypackage --registry https://custom-registry.com
update Dependencies

Update all dependencies to their latest compatible versions according to your version constraints.

zepplin update
build Build

Build your project with all dependencies resolved. This ensures all packages are downloaded and configured before running zig build.

zepplin build
publish Publishing

Publish your package to the registry. Requires authentication and a valid build.zig.zon with package metadata.

zepplin publish

Before publishing, ensure your build.zig.zon contains:

  • .name - Package name
  • .version - Semantic version (e.g., "1.0.0")
  • .description - Brief package description
  • .license - License identifier (e.g., "MIT")
login Authentication

Authenticate with a package registry. Required before publishing packages.

zepplin login [registry]

Arguments

[registry] Registry URL (default: https://registry.zepplin.dev)
discover Search

Search for packages across configured registries and Zigistry. Shows package details including stars and description.

zepplin discover [query]

Arguments

[query] Search query (searches name and description)

Examples

zepplin discover "http server"
zepplin discover json
zepplin discover
browse Discovery

Browse packages by category. Without a category, shows all available categories.

zepplin browse [--category <name>]

Options

--category <name> Filter by category (e.g., "cli", "web", "gamedev")

Examples

zepplin browse
zepplin browse --category=cli
zepplin browse --category=web
trending Discovery

Show trending packages based on recent activity, downloads, and community engagement.

zepplin trending [--category <name>]

Options

--category <name> Filter trending packages by category
help Help

Display help information and available commands.

zepplin help
